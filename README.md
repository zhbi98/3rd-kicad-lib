## 封装命名规则

(1) 使用下划线 `_` 分割不同描述类别。

(2) 使用横杆 `-` 分割相同描述类别的不同属性。

(3) 例如 `KEY-SMD_L4.2-W3.2-V_SKRPABE010`。

注意 L4.2, W3.2 尺寸不包含引脚长度，属于纯粹芯片封装外壳尺寸。
